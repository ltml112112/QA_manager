이 도구는 무엇인가요?

Agilent LC/MS Single Injection Report PDF의 2페이지에 있는 MSD 테이블 형식을 변환합니다.

RT / Type / Width / Area
Height / Area% / Name
RT / Type / m/z / Abun(%)
(Excel 데이터로 교체)

① Excel 파일 준비

아래 형식으로 작성합니다. A열=#, B열=m/z, C열=Abundance 순서를 지켜야 합니다.

A열 B열 (m/z) C열 (Abundance) 설명
#m/zAbundance그룹 구분 헤더행
37665.432.3피크 1 데이터
38666.562
………
#m/zAbundance그룹 구분 헤더행
21889.539.2피크 2 데이터
⚠ 피크(RT)가 1개인 경우 헤더행 1개 + 데이터만 입력
⚠ m/z 값은 소수점 첫째 자리로 자동 반올림됩니다 (예: 650 → 650.0)

② 파일명 규칙

PDF와 Excel의 파일명(확장자 제외)이 동일해야 자동으로 매칭됩니다.

✓ LT-PHM220_0409.pdf + LT-PHM220_0409.xlsx
✗ LT-PHM220_0409.pdf + LT-PHM220.xlsx

여러 쌍을 한꺼번에 업로드하면 매칭된 쌍만 순서대로 처리됩니다.

③ 사용 방법

1

업로드 영역에 PDF 파일과 Excel 파일을 함께 드래그하거나, 클릭 후 다중 선택합니다.

2

파일 매칭 결과를 확인합니다. ✓ 매칭 표시된 쌍만 처리됩니다.

3

선택된 쌍 모두 변환 & 다운로드 버튼을 클릭합니다.

✓

변환된 PDF가 원본파일명_수정본.pdf 형태로 자동 다운로드됩니다.

주의 사항

• Agilent Single Injection Report 형식의 2페이지 PDF에만 동작합니다.

• 인터넷 연결이 필요합니다 (라이브러리 로딩용).

• 파일은 서버로 전송되지 않으며 브라우저 내에서만 처리됩니다.

• Excel 1번째 시트의 데이터만 읽습니다.
