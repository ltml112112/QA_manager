# QA Manager 포털 — 기능 설명

전자재료사업부 품질경영팀 업무 자동화 포털.
좌측 사이드바에서 탭을 클릭해 각 도구로 이동합니다.

> **탭 잠금 안내**: 일부 탭(🔒)은 접근 코드 입력 후 사용 가능합니다.
> 사이드바 하단 초록 상태 점(●)을 클릭 → 코드 입력 → 잠긴 탭 전체 표시.
> 코드 입력 후에는 점을 다시 클릭해 공개 탭만 표시/전체 표시를 토글할 수 있습니다.

## 1. OLED IVL & LT data 시각화

**용도**: REF 소재와 SAMPLE 소재의 IVL·수명 데이터를 비교 분석하고, 결과를 양산 소자평가 Lot test 정리 엑셀에 붙여넣을 수 있도록 복사합니다.

**사용 흐름**
1. REF/SAMPLE 각각의 IVL, LT CSV 파일을 드래그앤드롭 (최대 8개, 파일명으로 슬롯 자동 배정)
2. **분석 실행** → J=10mA/cm² 기준으로 전압·효율·EQE·CIE·파장 자동 추출
3. REF vs SAMPLE 비교 테이블 확인 (±5% 이내 파란색 / 초과 빨간색 / 105% 초과 보라색)
4. LT 수명 곡선 차트 확인
5. **요약 버튼** → REF + SAMPLE 핵심 2행만 클립보드 복사 → 엑셀에 바로 붙여넣기
   - COPY 버튼으로 테이블 전체 복사도 가능
   - COPY CHART로 그래프 이미지 복사 가능

**참고**
- LT 모드: Low(LT90~95) / High(LT96~99) 전환 가능
- 최근 세션 최대 10개 자동 저장 (IndexedDB)

## 2. 소자평가 Lot 일정 관리

**용도**: 합성생산·정제생산/소자이관 부서의 소자평가 이관 일정을 월별 캘린더로 관리합니다. Firebase 실시간 동기화로 여러 사용자가 동시에 사용해도 즉시 반영됩니다.

**화면 구성**
- **상단 바**: 📧 메일 일괄 등록 / ✏ 개별 등록 / 월 네비게이션 / 오늘 / 🔍 조회
- **요약 스트립**: 진행 중·합성·정제소자·시급·완료 건수 칩 (클릭 시 해당 항목만 모달로 조회)
- **범례 바**: 합성생산(빨간색) · 정제소자이관(보라색) · 시급 요청 · 완료 + 완료 표시 토글
- **캘린더**: 7열 월별 그리드 — 셀당 최대 3개 카드, 초과 시 `+N개` 클릭으로 전체 조회

**캘린더 표시 규칙**
| 부서 | 표시 기간 |
|------|----------|
| 합성생산 | 이관 당일만 표시 |
| 정제생산/소자이관 | 이관일부터 완료일(또는 오늘)까지 모든 날짜에 표시 |

**등록 방법 — ① 메일 일괄 등록**
1. 📧 메일 일괄 등록 버튼 클릭
2. 이메일 본문의 표를 복사(Ctrl+C)한 뒤 팝업 영역에 붙여넣기(Ctrl+V) — 합성생산/정제소자이관 표 자동 판별
3. 미리보기에서 내용 확인 (편집 모드 토글 가능)
4. **일괄 등록** 버튼으로 한 번에 저장

> 합성생산 표는 같은 Batch No. 행이 그룹으로 묶여 배치당 1건으로 등록됩니다.

**등록 방법 — ② 개별 등록**
1. ✏ 개별 등록 버튼 클릭
2. 부서·재료명·Lot번호·요청사항·이관일 등 입력
3. **저장** → 캘린더에 즉시 반영

**날짜 클릭 / 조회 모달**
- 날짜 셀 클릭 → 해당 날 활성 항목을 합성생산(왼쪽) / 정제소자이관(오른쪽) 2컬럼으로 표시
- 오늘 셀 클릭 → 오늘 이관(합성) + 전체 미완료(정제소자) 목록
- 🔍 조회 버튼 → 품명·Lot·요청사항·부서·비고 통합 검색 (Enter 지원)
- 카드에서 수정·완료 처리·삭제 가능

## 3. HPLC/DSC Report 자동화 🔒

**용도**: 분석 보고서 PDF에서 HPLC·DSC 그래프를 잘라내어 PPT 슬라이드로 자동 생성합니다.

**사용 흐름**
1. HPLC + DSC 통합 분석 보고서 PDF 업로드
2. 오른쪽 미리보기를 보면서 슬라이더로 크롭 영역·폭·배율 미세조정
3. **다운로드 버튼** → HPLC / DSC 각각 별도 PPT 슬라이드 파일로 저장

## 4. LGD 사전심사자료 자동화 🔒

**용도**: LGD 사전심사 제출용 문서(PDF 5종 + Excel 2종)를 폼 입력만으로 자동 생성합니다.

**사용 흐름**
1. 작성일, 제품명, 색상, 상품명(1~3) 입력
2. **생성 버튼** → Google Sheets 템플릿에 값 치환 후 7개 파일 자동 생성
3. 생성된 파일 개별 다운로드 또는 ZIP 일괄 다운로드

**생성 파일 목록**

| 파일 | 형식 |
|------|------|
| MSDS | PDF, Excel |
| 경고표지 | PDF |
| 구성제품확인서 | PDF |
| 작업공정별관리요령 | PDF |
| 비공개물질확인서 + Checksheet | PDF, Excel |

## 6. Lot 추적관리 & SQC 🔒

**용도**: 소재 Lot의 완제품→원료 계보를 시각화하고, 각 Lot의 공정별 품질 데이터 TREND를 SPC 차트로 확인합니다.

**입력 파일**
- **흐름도 데이터** (Excel): Lot 간 투입 관계 (완제품↔원료 계보)
- **품질 데이터** (Excel): Batch별 공정 측정값 + USL/LSL/UCL/LCL 기준

> 흐름도 파일만 올려도 계보도는 즉시 표시됩니다. 품질 파일은 추가 시 SPC 차트가 함께 표시됩니다.

**사용 흐름**

**① Lot 계보 확인**
1. 흐름도 데이터 업로드
2. 좌측 사이드바에서 Lot 검색 또는 단계 필터 탭(완제품/정제1차품/원재료 등)으로 탐색
3. Lot 클릭 → 우측에 완제품→원료 방향 계보도 표시
4. **⎘ 계보 복사** → Excel에 붙여넣기 (같은 Lot는 병합 셀로 자동 처리)

**② SPC TREND 확인**
1. 품질 데이터 추가 업로드
2. 공정단계별 측정항목 SPC 차트 자동 표시 (UCL/LCL·USL/LSL 기준선 포함)
3. 차트의 포인트 클릭 → **Batch Deep-Dive** 패널에서 해당 Batch의 전 공정 측정값 일괄 확인

**③ 품질 데이터 추적 요약**
- Lot 선택 시 해당 Lot 및 연관 Lot 전체의 측정값 이탈 여부를 테이블로 요약

**사이드바 검색 범위**: Lot번호, 비고 LOTNUM 모두 검색 가능
